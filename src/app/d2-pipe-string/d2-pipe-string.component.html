2D. Pipe string


{{str}} ---- строка, как есть


{{str | uppercase}} ---- применяем пайп uppercase


{{str | lowercase}} ---- применяем пайп lowercase


{{str | uppercase | lowercase}} ---- применяем пайп uppercase | lowercase (сразу два)


{{str | titlecase}} ---- применяем пайп titlecase


{{str | slice:1:5}} ---- применяем пайп slice:1:5
